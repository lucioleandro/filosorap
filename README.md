## O que é isto?
Projeto feito nas horas vagas sem auxílio de bibliotecas e frameworks, utilizando apenas Vanilla JS, HTML e CSS, com o intuito de juntar punchlines divertidas, filosóficas ou criativas de músicas de rap.
